ABI-Bench v0.1

v0.1 pre-registered 预注册 2026-06-17 Agent-Bioinformatics Interface Benchmark — Evaluating whether a structured ABI control layer improves LLM agent operation of bioinformatics workflows. 智能体-生物信息学接口基准（ABI-Bench） — 评估结构化 ABI 控制层 是否能提升大语言模型智能体在生物信息学工作流中的操作能力。

Core question: Under the same LLM, same agent harness, same repository, same tasks, and same fixtures, does an ABI control layer enable agents to more reliably plan, dry-run, inspect, diagnose, recover, and report on bioinformatics workflows compared to README + Shell or Plain Tool Calling?
核心问题：在相同大语言模型、相同智能体框架、相同代码仓库、相同任务和相同测试夹具的条件下， ABI 控制层 是否能让智能体在生物信息学工作流中更可靠地完成规划、空运行、检查、诊断、恢复和报告， 相比 README + Shell 或 纯工具调用 两种基线？

7. Reproduce & Submit 7. 复现与提交

Quick Start (3 minutes, no API key) 快速开始 (3 分钟，无需 API Key) 

# Install ABI# 安装 ABI
pip install abi

# Clone benchmark# 克隆基准仓库
git clone <this-repo-url> ABI-Bench && cd ABI-Bench

# Validate infrastructure (simulated mode — no LLM needed)# 基础设施验证 (模拟模式 — 无需 LLM)
python bench/harness/run_task.py --group G3 --task T03 --agent-mode simulated
python bench/scoring/score_run.py \
  --task bench/tasks/T03_dryrun_plasmid.yaml \
  --trace-dir bench/traces/G3/T03/replicate_01 \
  --run-dir bench/workspaces/G3/T03/replicate_01 \
  --output bench/results/G3/T03/replicate_01/score.json
# Expected: 12/12# 预期: 12/12

Run Full Benchmark 运行完整基准 

# Copy environment config# 复制环境配置
cp bench/.env.example bench/.env
# Edit bench/.env — set your API key and model# 编辑 bench/.env — 设置你的 API Key 和模型

# Run all 3 groups (MVP 8 tasks, 3 replicates)# 运行全部 3 组 (MVP 8 任务, 3 次重复)
for group in G1 G2 G3; do
  ABI_BENCH_MAX_TOKENS=8000 python bench/harness/run_group.py \
    --group $group --tasks mvp --replicates 3 \
    --agent-mode direct --parallel --workers 4 \
    --experiment-set main --fixture-set public \
    --outdir bench/results/$group
done

# Aggregate & score# 聚合与评分
python bench/scoring/aggregate_scores.py \
  --results bench/results --experiment-set main --fixture-set public \
  --output bench/results/leaderboard.tsv --summary bench/results/summary.json

# Run preflight check# 运行预检
python bench/scoring/claim_preflight.py \
  --results bench/results --experiment-set main --fixture-set public \
  --min-replicates 3
